# 🎨 COOS — Creative Order & Online Studio System

COOS is a web application for managing an online creative studio.
It allows customers to browse artwork, place orders, and track progress, while admins and editors manage production workflows.

---

## 🚀 Features

* 🔐 User Authentication (Register / Login)
* 🧩 Work Types & Packages Management
* 🖼 Gallery Showcase (filter by type & tags)
* 📜 Policies (Terms, Refund, Privacy)
* 🛒 Order & Payment System *(coming soon)*
* 🎨 Editor Workflow *(coming soon)*
* 📊 Dashboard & Reports *(coming soon)*

---

## 🏗 Tech Stack

* **Frontend:** Nuxt.js
* **Backend:** Node.js / Express
* **Database:** MySQL

---

## 📂 Project Structure

```
/backend   → API & Database
/frontend  → User Interface (Next.js)
```

---

## 🧠 Project Concept

This system is designed to simulate a real-world digital art service platform where:

* Customers can select styles and packages
* Admin manages system data
* Editors receive and produce work
